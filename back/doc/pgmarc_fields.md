# Tags present in the Project Gutenberg MARC Records

Below is a list of MARC record tags present in the Project Gutenberg MARC-XML
file, mapped with their descriptions from the [www.loc.gov](www.loc.gov) website.

[x] 001: Control Number

[] 003: Control Number Identifier

[] 006: Fixed-Length Data Elements - Additional Material Characteristics

[] 007: Physical Description Fixed Field

[] 008: Fixed-Length Data Elements

[] 010: Library of Congress Control Number (NR)

[x] 020: International Standard Book Number (R)

[] 040: Cataloging Source (NR)

[x] 041: Language Code (R)

[x] 050: Library of Congress Call Number (R)

[x] 100: Main Entry - Personal Name (NR)

[x] 240: Uniform Title (NR)

[x] 245: Title Statement (NR)

[] 246: Varying Form of Title (R)

[] 250: Edition Statement (R)

[x] 264: Production, Publication, Distribution, Manufacture, and Copyright
Notice (R)

[] 300: Physical Description (R)

[] 336: Content Type (R)

[] 337: Media Type (R)

[] 338: Carrier Type (R)

[x] 490: Series Statement (R)

[x] 500: General Note (R)

[x] 505: Formatted Contents Note (R)

[x] 508: Creation/Production Credits Note (R)

[x] 520: Summary, etc. (R)

[x] 534: Original Version Note (R)

[x] 546: Language Note (R)

[x] 653: Index Term - Uncontrolled (R)

[x] 700: Added Entry - Personal Name (R)

[] 830: Series Added Entry - Uniform Title (R)

[x] 856: Electronic Location and Access (R)
